# Ecommerce-Review-Trust-Analyzer
A deep learning-powered web application that detects AI-generated fake reviews, analyzes customer sentiment, and generates concise feedback summaries for e-commerce merchants.
Live Demo: https://ecommerce-review-trust-analyzer-rxixvpadxecqjvqfa67pjz.streamlit.app/

Features
🔍 AI-Generated Review Detection – Identify computer-generated fake reviews using fine-tuned DistilBERT

😊 Sentiment Analysis – Classify authentic reviews as positive or negative with RoBERTa

📝 Review Summarization – Generate concise summaries of customer feedback with FLAN-T5

📊 Batch Analysis – Upload CSV files to analyze multiple reviews at once

💡 Business Insights – Get trust scores, sentiment distribution, and actionable recommendations
