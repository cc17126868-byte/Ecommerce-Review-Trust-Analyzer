# 📊 Review Trust & Customer Insight Analyzer

## 📌 Overview
This project develops an AI-powered system to analyze customer reviews in e-commerce environments.

It focuses on:
- Detecting **computer-generated reviews** (as a form of low-quality or potentially misleading content)
- Performing **sentiment analysis**
- Generating **concise review summaries**

The system is designed for Shopify merchants to improve review trustworthiness and extract actionable insights.

---

## 🚀 Key Features

- 🔍 **Computer-Generated Review Detection**  
  Identify reviews likely generated by AI or automated systems  

- 😊 **Sentiment Analysis**  
  Classify reviews as positive or negative  

- 🧠 **Review Summarization**  
  Generate short summaries for quick understanding  

- 📊 **Dataset Analysis Dashboard**  
  Visualize authenticity and sentiment distribution  

- 📁 **CSV Upload Support**  
  Analyze large datasets efficiently

## 🤖 Models Used

### 1️⃣ Computer-Generated Review Detection
- Model: `distilbert-base-uncased` (fine-tuned)
- Task: Binary Classification  

**Labels:**
- `LABEL_0` → Human-written  
- `LABEL_1` → Computer-generated  

🔗 https://huggingface.co/JerryJJJJJ/fake-review-detector-distilbert  

---

### 2️⃣ Sentiment Analysis
- Model: `roberta-base` (fine-tuned)
- Task: Sentiment Classification  

**Labels:**
- `LABEL_0` → Negative  
- `LABEL_1` → Positive  

🔗 https://huggingface.co/JerryJJJJJ/sentiment-amazon-roberta  

---

### 3️⃣ Review Summarization
- Model: `google/flan-t5-base` (fine-tuned)
- Task: Text Summarization  

🔗 https://huggingface.co/JerryJJJJJ/review-summarization-flan-t5
